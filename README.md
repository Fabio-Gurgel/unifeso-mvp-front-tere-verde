# Projeto da disciplina MVP Front-end - Unifeso
![Tela inicial](./prototipo/banner-readme.jpg)
## 🚵 Objetivos e benefícios do produto

Desenvolver uma aplicação web que centralize e facilite o acesso a informações sobre o Parque Nacional da Serra dos Órgãos, o Parque Estadual dos Três Picos e o Parque Natural Municipal Montanhas de Teresópolis, proporcionando uma experiência de consulta simples, intuitiva e eficiente para visitantes e interessados.

A plataforma permitirá a visualização de informações relevantes, como localização, imagens, horários de funcionamento e condições de visitação, além de conteúdos relacionados a trilhas, cachoeiras, eventos e biodiversidade (fauna e flora). Dessa forma, o sistema contribui para melhorar o planejamento das visitas e ampliar o conhecimento dos usuários sobre os atrativos naturais da região.

## 🏹 Público alvo e suas dores

O sistema é voltado para pessoas interessadas em turismo ecológico e atividades ao ar livre na região de Teresópolis, incluindo:

- Turistas ocasionais em busca de lazer e contato com a natureza
- Praticantes de trilhas e ecoturismo
- Estudantes e entusiastas de biodiversidade
- Moradores da região que desejam explorar melhor os atrativos locais

### Principais dores
- Informação descentralizada: dados sobre parques, trilhas e atrações estão espalhados em diferentes sites e redes sociais, muitas vezes desatualizados.
- Dificuldade no planejamento: falta de informações claras sobre acesso, horários, nível de dificuldade e condições das trilhas.
- Baixa visibilidade da biodiversidade local: pouca divulgação organizada sobre fauna e flora da região.
- Falta de confiabilidade: conteúdos informais ou incompletos dificultam a tomada de decisão do visitante.
- Experiência fragmentada: o usuário precisa consultar múltiplas fontes para montar um roteiro simples.

## 👑 MVP Definido

O sistema será disponibilizado via web, com acesso gratuito e sem necessidade de autenticação para usuários visitantes, permitindo a consulta de conteúdos sobre parques de Teresópolis.

A plataforma permitirá a navegação pelos seguintes conteúdos:

- Parques
- Trilhas
- Cachoeiras
- Fauna
- Flora
- Eventos

Todas as entidades possuirão páginas de detalhamento com informações relevantes, como descrições, imagens e dados específicos de cada tipo.

### Parques
Informações como nome, descrição, localização, imagens, horário de funcionamento e associação com trilhas e cachoeiras.

### Trilhas
Dados como dificuldade, tempo estimado, elevação, horário recomendado, descrição e imagens.

### Cachoeiras
Informações como descrição, imagens, profundidade e indicação para banho.

### Flora
Dados como nome científico, descrição, uso medicinal e imagens.

### Fauna
Informações como nome popular e científico, descrição, status de conservação e imagens.

### Eventos
Dados como nome, descrição, data, localização e imagens.

### Área Administrativa
O sistema contará com uma área administrativa restrita, acessível mediante autenticação, permitindo o gerenciamento completo dos dados (criar, editar e excluir) para todas as entidades: parques, trilhas, cachoeiras, fauna, flora e eventos.

## 🍃 Como rodar o projeto?
- Node 22 instalado
- Clonar o projeto
- Dentro do projeto, na pasta /mvp-tere-verde, rodar os seguintes comandos:
  - npm i
  - npm run server para subir o Json Server
  - npm run dev para subir o projeto
